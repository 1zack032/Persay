Data Collection

What information we collect and why we need it.

Information You Provide

When you create an account and use Menza, you may provide us with:

Account Information

  • Username: Your chosen display name for the platform
  • Email Address: For account recovery and important notifications
  • Password: Securely hashed and stored (we never store plain text passwords)
  • Age Verification: Confirmation that you are 18 years or older

Profile Information

  • Profile picture (optional)
  • Display name (optional)
  • Bio or status message (optional)

Communication Data

  • Messages: End-to-end encrypted; we cannot read message content
  • Media: Photos, videos, and files you share (encrypted)
  • Channel content: Posts you create in public or private channels

Information Collected Automatically

When you use Menza, we automatically collect certain technical information:

Device Information

  • Device type (mobile, desktop, tablet)
  • Operating system and version
  • Browser type and version
  • Screen resolution

Usage Information

  • Login timestamps
  • Features used
  • Error logs and crash reports
  • Performance metrics

Connection Information

  • IP address (temporarily, for security purposes)
  • Connection timestamps
  • General geographic location (country/region level only)

Legal Basis for Collection

Under GDPR, we collect data based on:

  • Contract: To provide you with our messaging service
  • Legitimate Interest: To improve security and prevent abuse
  • Consent: For optional features like analytics
  • Legal Obligation: To comply with applicable laws
